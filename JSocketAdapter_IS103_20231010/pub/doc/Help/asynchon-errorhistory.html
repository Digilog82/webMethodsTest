KFTC Online > Online Error History Screen
Screen Heading: IS Administrator > Solutions > Socket Admin > KFTC Online > Online Error History
Provider System Name이 'KFTC'인 Asynch Online Socket 인터페이스 실행중에 에러가 발생한 내역을 조회하는 메뉴이다.
  • KFTC Online > Online Error History 클릭 : 목록을 조회한다.
  • Delete 아이콘 클릭 : 해당 에러 내역을 삭제한다.
  • Delete All 클릭 : KFTC의 모든 에러 내역을 삭제한다. 에러 내역이 존재할 경우에만 이 버튼이 Display 된다.
Search Screen
Note:  
  • KFTC는 예를 들어서 설명하기 위한 System Name 이다.
  • Basic Info > Business Name 메뉴에서 Asynch Online Socket YN 필드가 'Yes'인 Business Name이 존재할 경우 Provider System Name 단위로 "ProviderSystemName Online" 이라는 메뉴가 생성된다.
  • Asynch Online Socket 인터페이스 실행중에 에러가 발생한 경우 원인 분석하는 데에 활용할 수 있다.
  • 파일 저장 위치 : SAG_Directory/CUSTOM_CONFIG/socket/online/error
  • Basic Info > Delete Local File 메뉴에서 위의 디렉토리를 등록해서 보관일수가 지난 파일은 자동으로 삭제되도록 할 수 있다.

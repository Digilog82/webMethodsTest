Utility > Server Resources Screen
Screen Heading: IS Administrator > Solutions > Socket Admin > Utility > Server Resources
ESB 서버의 Memory, CPU, Thread Pool 등 Resources 상태를 조회하는 메뉴이다.
  • Utility > Server Resources 클릭 : 조회화면으로 이동한다.
  • Select Server : IS Administrator > Settings > Remote Servers 메뉴에서 등록한 목록이 Display 되고 버튼을 클릭하면 해당 Remote Server의 Resources 상태를 조회한다.
  • Refresh Type : Manual(관리자가 직접 클릭해서 Resources 상태를 다시 조회), Auto(자동으로 Resources 상태를 다시 조회)
    Auto로 설정한 경우에는 Auto Refresh Interval을 초단위로 입력 후 Refresh 버튼을 클릭하면 Interval 간격으로 화면이 자동으로 Refresh 되어 Resources 상태를 조회한다.
Search Screen
Note:  
  • ESB 서버의 IS Heap Memory 사용현황, Service Instance 및 Thread Pool 사용현황, Hardware의 CPU, Memory 사용현황 및 Java Process 목록을 한 화면에서 조회할 수 있다.
  • Resources 상태는 화면에 진입하는 순간의 상태로서 지속적으로 모니터링하기 위해서는 화면을 Refresh 해야 한다.

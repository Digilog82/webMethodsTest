KFTC Online > Data Q Listener Screen
Screen Heading: IS Administrator > Solutions > Socket Admin > KFTC Online > Data Q Listener
Asynch Online Socket Client에 대한 AS400 Data Queue Listener 목록을 조회하는 메뉴이다.
  • KFTC Online > Data Q Listener 클릭 : 목록을 조회한다.
Search Screen
Note:  
  • KFTC는 예를 들어서 설명하기 위한 System Name 이다.
  • Basic Info > Business Name 메뉴에서 Asynch Online Socket YN 필드가 'Yes'인 Business Name이 존재할 경우 Provider System Name 단위로 "ProviderSystemName Online" 이라는 메뉴가 생성된다.
  • KFTC Online > Data Q Listener 메뉴는 AS400 시스템이 존재하면서 Data Queue를 이용해서 Asynch Online 방식의 Socket 인터페이스를 위해서 ESB와 연계하는 경우에만 Display 된다.
    즉, Basic Info > Custom Variable 메뉴에서 ESB.CONFIG.AS400.ASYNCHONLINEYN의 Value가 true로 설정되어 있는 경우에만 Display 된다.
  • AS/400 > Data Q Listener 메뉴에 등록되어 있는 Listeners 중에서 Provider System Name이 'KFTC'이고 Asynch Online Socket YN이 'Yes'인 Listeners 목록이 조회되고 그리고 Listener의 Enable 상태를 조회하기 위한 것이다.
  • Data Q Listener는 Asynch Online Socket Business Name 단위로 등록한다.
  • 운영서버가 두 대인 경우 한 쪽 운영서버에서만 Enabled 되도록 해야 한다.

Utility > Suspend E-Mail Notification Screen
Screen Heading: IS Administrator > Solutions > Socket Admin > Utility > Suspend E-Mail Notification
ESB 관리자에게 메일로 통보하는 기능을 중지하는 Schedule을 등록, 관리하는 메뉴이다.
  • Utility > Suspend E-Mail Notification 클릭 : 목록을 조회한다.
  • Create Schedule 클릭 : Schedule을 등록한다. Schedule 등록을 하지 않은 상태에서는 Create Schedule 버튼이 Display 된다.
  • Update Schedule 클릭 : Schedule 실행시간을 수정한다. Schedule을 등록하고 Status가 Active인 경우에는 Update Schedule 버튼이 Display 된다. Start Date, Start Time, End Date, End Time을 수정하고 Update 한다.
Create And Update Screen
Note:  
  • JSocketAdapter에서 수행하는 눈에 보이지 않는 기능을 수행 시 수행결과 또는 에러발생에 대한 내용을 ESB 관리자에게 메일로 통보를 하는데 이 기능을 정해진 시간동안 중지시키는 Schedule을 등록하는 것이다.
  • Start Date, Start Time에 실행되는 Schedule 하나와 End Date, End Time에 실행되는 Schedule 하나가 등록된다.
    Start Date, Start Time에 실행되는 Schedule은 Basic Info > Custom Variable 메뉴의 ESB.CONFIG.INCHARGE.EMAILYN의 Value를 false로 설정하여 ESB 관리자에게 메일로 통보하지 않도록 한다.
    End Date, End Time에 실행되는 Schedule은 Basic Info > Custom Variable 메뉴의 ESB.CONFIG.INCHARGE.EMAILYN의 Value를 true로 설정하여 ESB 관리자에게 메일로 통보하도록 한다.
    즉, Start Date, Start Time ~ End Date, End Time 시간동안 메일로 통보하는 것이 중지된다.
  • 내외부 시스템 Shutdown 작업이 예정되어 있을 경우에 유용하게 사용할 수 있는 기능이다.
  • 여기에서 등록한 Schedule은 IS Administrator > Scheduler > One-Time and Simple Interval Tasks에서도 확인 가능하다. 이 Schedule들은 한 번만 실행되는 것으로서 실행되고 나면 목록에서 자동으로 삭제된다.

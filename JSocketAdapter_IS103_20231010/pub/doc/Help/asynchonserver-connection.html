Online Server > Port 7777 Screen
Screen Heading: IS Administrator > Solutions > Socket Admin > Online Server > Port 7777
Asynch Online Socket Server Port Number 7777에 대한 연결정보를 조회하고 Port [Open/Close]를 처리하는 메뉴이다.
  • Online Server > Port 7777 클릭 : 해당 Port Number에 대한 상세정보 및 연결되어 있는 Client에 대한 상세정보를 조회한다.
  • Server Run 클릭 : 해당 Port Number로 서버를 기동한다. 서버가 Shutdown 상태일 때 이 버튼이 Display 된다.
  • Server Down 클릭 : 해당 Port Number에 대해서 서버를 Shutdown 한다. 서버가 기동중일 때 이 버튼이 Display 된다.
Search Screen
Note:  
  • 7777은 예를 들어서 설명하기 위한 Port Number 이다.
  • Server Socket > Local Server Socket 메뉴에서 Asynch Online YN 필드가 'Yes'인 Port Number가 존재할 경우
    "Online Server" 라는 메뉴가 생성되고 그 아래에 "Port PortNumber" 라는 메뉴가 Port Number 별로 생성된다.
  • Asynch Online Socket Server Port Number로 Client 접속이 이루어지면 연결상태를 계속 유지를 한다. 현재 연결되어 있는 Client의 연결정보를 상세하게 조회할 수 있다.
  • 수신전용 Port와 송신전용 Port가 따로 있는 경우에는 두 Port에 대한 연결정보가 모두 조회된다.

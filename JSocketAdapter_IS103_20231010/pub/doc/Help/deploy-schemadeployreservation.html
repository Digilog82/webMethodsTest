Deploy Config > Doc Schema Reservation Screen
Screen Heading: IS Administrator > Solutions > Socket Admin > Deploy Config > Doc Schema Reservation
Deploy Config > Doc Schema Define 메뉴에서 Deploy 예약을 한 Schedule을 [조회/수정/삭제] 하는 메뉴이다.
이 메뉴에서 관리하는 Schedule은 IS Administrator > Server > Scheduler 메뉴에서도 확인할 수 있다.
  • Deploy Config > Doc Schema Reservation 클릭 : 목록을 조회한다.
  • Edit 아이콘 클릭 : 상세조회 및 수정화면으로 이동한다.
  • Delete 아이콘 클릭 : 등록정보를 삭제한다.
Search And Update Screen
Note:  
  • Deploy Config > Doc Schema Reservation 메뉴는 개발서버 또는 UAT서버에서만 Display 된다.
  • Deploy Config > Doc Schema Define 메뉴에서 "Reserve Deployment" 버튼을 클릭한 경우 Deploy 예약 Schedule이 Deploy Config > Doc Schema Reservation 메뉴에 등록된다.
  • Edit 하는 경우에는 Schedule 실행 날짜, 시간만 수정 가능하다.
  • 이 Schedule은 한 번만 실행되는 것으로서 실행되고 나면 목록에서 자동으로 삭제된다.

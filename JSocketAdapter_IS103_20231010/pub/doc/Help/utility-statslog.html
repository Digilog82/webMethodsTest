Utility > Stats Log Analysis Screen
Screen Heading: IS Administrator > Solutions > Socket Admin > Utility > Stats Log Analysis
Integration Server Stats Log 파일을 날짜별로 Download 해서 분석할 수 있는 메뉴이다.
  • Utility > Stats Log Analysis 클릭 : Download 화면으로 이동한다.
  • Download File 클릭 : 조회조건에 해당하는 Stats Log 파일을 Download 한다.
  • Open Stats Log Analyzer 클릭 : Download 한 Stats Log 파일에 대해서 분석할 수 있는 화면을 Open 한다.
Download Screen
Note:  
  • SAG_Directory/logs 아래에 있는 stats.log 파일을 날짜별로 Download 할 수 있다.
  • Download 한 Stats Log 파일에 대해서 분석하기 위해서 "Open Stats Log Analyzer" 버튼을 클릭하면 새로운 창이 Open 된다. 그 창에서 Download 한 Stats Log 파일을 열어서 분석 작업을 수행한다.
  • Stats Log 분석을 Internet Explorer에서 수행하면 IE 버전에 따라서 Javascript 오류 발생으로 인해서 정상적으로 분석이 되지 않을 수도 있다. 그럴 경우 Chrome에서 수행하면 정상적으로 분석이 된다.

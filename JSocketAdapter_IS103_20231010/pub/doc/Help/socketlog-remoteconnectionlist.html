Socket Log > Remote Server Connection Log Screen
Screen Heading: IS Administrator > Solutions > Socket Admin > Socket Log > Remote Server Connection Log
Online Remote Socket Server Socket Name에 대한 연결 및 연결해제 Log를 조회하는 메뉴이다.
  • Socket Log > Remote Server Connection Log 클릭 : 조회화면으로 이동한다.
  • Search Logs 클릭 : 조회조건에 해당하는 Connection Log 목록을 조회한다.
Search Screen
Note:  
  • Basic Info > Custom Variable 메뉴의 ESB.SOCKET.CONNLOG.SAVEYN의 Value가 true인 경우에만 로그파일로 저장된다.
  • Server Socket > Remote Server Socket 메뉴에서 Asynch Online YN 필드의 값이 Yes로 설정된 Socket Name에 대해서만 로그파일을 저장한다.
  • 로그파일은 날짜별 폴더 아래에 생성된다. 로그파일 Naming Rule은 아래와 같다.
    Socket Name + "$" + 발생시간 + "$" + Action(Connect 또는 Close) + "$" + serverIP + "$" + Status(Success 또는 Fail) + "$" + UUID
  • 파일 저장 위치
    SAG_Directory/CUSTOM_CONFIG/socket/cconnlogs
    운영서버의 경우 NAS 등 공유 디렉토리가 있는 경우에는 공유 디렉토리를 설정해야 로그가 분산되어 저장되지 않는다.
  • Basic Info > Delete Local File 메뉴에서 위의 디렉토리를 등록해서 보관일수가 지난 파일은 자동으로 삭제되도록 할 수 있다.
  • 조회조건 중에서 Log Date, Socket Name은 반드시 입력해야 한다.
Log Date
로그파일이 생성된 날짜
Log Time
로그파일이 생성된 시간
Socket Name
조회하고자 하는 데이터 송수신 Socket에 대한 Name

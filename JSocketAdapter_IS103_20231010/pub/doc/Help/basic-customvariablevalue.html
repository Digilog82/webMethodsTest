Basic Info > Custom Variable Value Screen
Screen Heading: IS Administrator > Solutions > Socket Admin > Basic Info > Custom Variable Value
Basic Info > Custom Variable 메뉴에서 관리하는 Variable Name에 대한 Value가 두 개 이상의 고정된 Values 중에서 선택해서 셋팅해야 하는 경우 Variable Name에 대한 Values를 [등록/수정/삭제/조회] 하는 메뉴이다.
JSocketAdapter Package를 최초에 Install 하면 Custom Variable Value 기본값들이 자동으로 등록된다.
  • Basic Info > Custom Variable Value 클릭 : 목록을 조회한다.
  • Create New Custom Variable Value 클릭 : 신규정보 등록화면으로 이동한다.
  • Edit 아이콘 클릭 : 상세조회 및 수정화면으로 이동한다.
  • Delete 아이콘 클릭 : 등록정보를 삭제한다.
Create And Update Screen
Note:  
  • Variable Name은 기존에 등록되지 않은 Unique 한 값으로 해야 한다.
  • 모든 Field에 대해서는 필수로 입력하고 [Value/Description] 쌍이 두 개 이상 되어야 저장이 가능하다.
  • 운영서버가 두 대인 경우 한 쪽 운영서버에서 [등록/수정/삭제] 하면 다른 쪽 운영서버에서도 자동으로 [등록/수정/삭제] 된다.
  • Config File : SAG_Directory/CUSTOM_CONFIG/socket/customVariableValueConfig.xml

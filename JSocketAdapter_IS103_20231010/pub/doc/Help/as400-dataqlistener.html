AS/400 > Data Q Listener Screen
Screen Heading: IS Administrator > Solutions > Socket Admin > AS/400 > Data Q Listener
IS Administrator > Adapters > AS/400 Adapter > Data Queue Listeners 메뉴에 등록되어 있는 Listeners를 [등록/수정/삭제/조회] 하는 메뉴이다.
  • AS/400 > Data Q Listener 클릭 : 목록을 조회한다.
  • Register New Data Q Listener 클릭 : 신규정보 등록화면으로 이동한다.
  • Edit 아이콘 클릭 : 상세조회 및 수정화면으로 이동한다.
  • Delete 아이콘 클릭 : 등록정보를 삭제한다.
Create And Update Screen
Note:  
  • AS/400 메뉴는 AS400 시스템이 존재하고 Data Queue를 이용해서 ESB와 연계하는 경우에만 Display 된다.
    즉, Basic Info > Custom Variable 메뉴에서 ESB.CONFIG.AS400.DATAQUSEYN의 Value가 true로 설정되어 있는 경우에만 Display 된다.
  • AS/400 메뉴는 IS Administrator > Adapters > AS/400 Adapter에서 제공되지 않는 Data Queue Listener 관리에 대한 편리 기능을 제공한다.
    이런 기능을 사용하기 위해서는 IS Administrator > Adapters > AS/400 Adapter > Data Queue Listeners 메뉴에서 등록한 Listener는 AS/400 > Data Q Listener 메뉴에서도 모두 등록하도록 한다.
  • Data Queue Listener를 이용해서 ESB 시스템의 Asynch Online Socket 인터페이스를 사용하는 경우에 AS/400 > Data Q Listener 메뉴에서 등록한 Listeners 정보를 이용해서 Asynch Online Socket 관리를 한다.
  • Listener Node Name은 기존에 등록되지 않은 Unique 한 값으로 해야 한다.
  • Listener Node Name은 IS Administrator > Adapters > AS/400 Adapter > Data Queue Listeners 메뉴에 등록되어 있는 Listeners 중에서 선택하도록 되어 있다.
  • Provider System Name 선택에 따라서 해당 System에 관련된 Business Name 목록이 자동으로 Display 된다.
  • Business Name 선택에 따라서 Asynch Online Socket YN이 자동으로 선택된다.
  • 모든 Field에 대해서는 필수로 입력한다.
  • 운영서버가 두 대인 경우 한 쪽 운영서버에서 [등록/수정/삭제] 하면 다른 쪽 운영서버에서도 자동으로 [등록/수정/삭제] 된다.
  • Config File : SAG_Directory/CUSTOM_CONFIG/as400/dataqListenerConfig.xml
Listener Node Name
IS Administrator > Adapters > AS/400 Adapter > Data Queue Listeners 메뉴에 등록되어 있는 Listeners 중에서 선택
Provider System Name
Listener가 이용하는 ESB 인터페이스의 Target System Name을 선택
Business Name
Listener가 이용하는 ESB 인터페이스의 Business Name을 선택
Asynch Online Socket YN
Listener가 이용하는 ESB 인터페이스가 Asynch Online Socket 인터페이스인지 여부
Asynch Online Socket YN이 'Yes' 인 Listener가 ProviderSystemName Online > Data Q Listener 메뉴에서 Display 되고 관리된다.

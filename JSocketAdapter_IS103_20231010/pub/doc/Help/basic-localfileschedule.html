Basic Info > Local File Delete Schedule Screen
Screen Heading: IS Administrator > Solutions > Socket Admin > Basic Info > Local File Delete Schedule
ESB 서버 Local 파일 시스템에 쌓이는 파일의 주기적인 삭제를 위한 Schedule을 [등록/수정/조회] 하는 메뉴이다.
이 메뉴에서 관리하는 Schedule은 IS Administrator > Server > Scheduler 메뉴에서도 확인할 수 있다.
  • Basic Info > Local File Delete Schedule 클릭 : 목록을 조회한다.
  • Create Schedule 버튼 클릭 : Schedule을 등록한다. Schedule 등록을 하지 않은 상태에서는 Create Schedule 버튼이 Display 된다.
  • Suspend Schedule 버튼 클릭 : Schedule을 중지한다. Schedule을 등록하고 Status가 Active인 경우에는 Suspend Schedule 버튼이 Display 된다.
  • Active Schedule 버튼 클릭 : Schedule을 Active 시킨다. Schedule을 등록하고 Status가 Suspended인 경우에는 Active Schedule 버튼이 Display 된다. Run Time을 수정하고 Active 시킬 수 있다.
Create And Update Screen
Note:  
  • Schedule Service는 JSocketAdapter.COMMON.SCHEDULE:deleteLocalFiles로 고정값이다.
  • 파일 삭제처리를 하기 위해서는 Basic Info > Delete Local File 메뉴에서 파일삭제 디렉토리 정보를 등록해야 한다.
  • Schedule은 하루에 한 번 실행된다.
ID

Service

Utility > Upload File Screen
Screen Heading: IS Administrator > Solutions > Socket Admin > Utility > Upload File
PC Local 파일시스템에 존재하는 파일을 ESB Local 파일시스템에 Upload 하는 메뉴이다.
  • Utility > Upload File 클릭 : Upload 처리 화면으로 이동한다.
  • Search File 클릭 : Upload Path에 있는 파일 목록을 조회한다.
  • Upload File 클릭 : File Upload 처리를 한다.
  • Delete Files 버튼 클릭 : 선택한 파일들을 ESB Local 파일시스템에서 삭제한다.
  • Delete 버튼 클릭 : 해당 파일을 ESB Local 파일시스템에서 삭제한다.
Upload Screen
Note:  
  • PC의 Local 파일시스템에 존재하는 파일을 ESB Server Local 파일시스템에 Upload 할 수 있다.
    PC에서 ESB Server에 FTP 또는 SFTP 연결이 안 되는 경우에 유용하게 사용할 수 있다.
  • 파일을 여러 개 선택하여 동시에 Upload 할 수도 있다.
  • ESB Server Local 파일시스템에 존재하는 파일을 삭제할 수 있다.
  • Check Box를 두 개 이상 선택해서 "Delete Files" 버튼을 클릭하면 선택된 파일들을 동시에 삭제할 수 있다.
  • Delete 버튼을 클릭하면 해당 파일 하나만 삭제한다.

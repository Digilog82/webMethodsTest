Online Server > Online Server List Screen
Screen Heading: IS Administrator > Solutions > Socket Admin > Online Server > Online Server List
Online Local Socket Server의 Port Number에 대한 정보를 조회하고 Port를 Enable 및 Disable 처리를 하는 메뉴이다.
  • Online Server > Online Server List 클릭 : 목록을 조회한다.
  • Search 클릭 : 조회조건에 맞는 정보를 조회한다.
  • Enabled 필드의 링크 클릭 : 해당 Port를 Enable/Disable 한다.
  • Port Number 필드의 링크 클릭 : 해당 Port Number에 대한 상세정보 및 연결되어 있는 Client에 대한 상세정보를 조회한다.
  • Port Enable 클릭 : 선택한 Port 또는 조회된 모든 Port를 일괄적으로 Enable 한다.
  • Port Disable 클릭 : 선택한 Port 또는 조회된 모든 Port를 일괄적으로 Disable 한다.
Search Screen
Note:  
  • Custom Variable "ESB.SOCKET.ONLINE.MENUDISPLAY"가 inlist로 설정된 경우에 이 메뉴가 Display 된다.
  • Local Server Socket Port 중에서 Online Port에 대해서만 조회를 하고 Port Enable 및 Disable 처리를 한다.
  • 체크박스를 선택하는 경우에는 선택된 Port에 대해서만 일괄적으로 Port Enable 및 Disable 처리를 한다.
  • 체크박스를 전체 선택하든가 아예 선택하지 않으면 조회된 모든 Port에 대해서 일괄적으로 Port Enable 및 Disable 처리를 한다.
  • 운영서버가 두 대인 경우 한 쪽 운영서버에서 [Enable/Disable] 하면 다른 쪽 운영서버에서도 자동으로 [Enable/Disable] 된다.

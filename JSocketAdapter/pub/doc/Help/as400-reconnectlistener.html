AS/400 > Reconnect Listener Screen
Screen Heading: IS Administrator > Solutions > Socket Admin > AS/400 > Reconnect Listener
IS Administrator > Adapters > AS/400 Adapter > Data Queue Listeners 메뉴에 등록되어 있는 Listeners에 대해서 연결상태를 정상적으로 유지하기 위한 Schedule을 [등록/수정/조회] 하는 메뉴이다.
이 메뉴에서 관리하는 Schedule은 IS Administrator > Server > Scheduler 메뉴에서도 확인할 수 있다.
  • AS/400 > Reconnect Listener 클릭 : 목록을 조회한다.
  • Create Schedule 버튼 클릭 : Schedule을 등록한다. Schedule 등록을 하지 않은 상태에서는 Create Schedule 버튼이 Display 된다.
  • Suspend Schedule 버튼 클릭 : Schedule을 중지한다. Schedule을 등록하고 Status가 Active인 경우에는 Suspend Schedule 버튼이 Display 된다.
  • Active Schedule 버튼 클릭 : Schedule을 Active 시킨다. Schedule을 등록하고 Status가 Suspended인 경우에는 Active Schedule 버튼이 Display 된다. Interval을 수정하고 Active 시킬 수 있다.

Note:  
  • AS/400 메뉴는 AS400 시스템이 존재하고 Data Queue를 이용해서 ESB와 연계하는 경우에만 Display 된다.
    즉, Basic Info > Custom Variable 메뉴에서 ESB.CONFIG.AS400.DATAQUSEYN의 Value가 true로 설정되어 있는 경우에만 Display 된다.
  • AS/400 메뉴는 IS Administrator > Adapters > AS/400 Adapter에서 제공되지 않는 Data Queue Listener 관리에 대한 편리 기능을 제공한다.
    이런 기능을 사용하기 위해서는 IS Administrator > Adapters > AS/400 Adapter > Data Queue Listeners 메뉴에서 등록한 Listener는 AS/400 > Data Q Listener 메뉴에서도 모두 등록하도록 한다.
  • Listener를 Enable 시키면 AS400 시스템과 Socket 연결이 되고 Disable 시키기 전까지는 연결상태를 계속 유지한다.
    그런데 AS400 시스템에서 정기 PM작업 등에 의해서 Data Queue를 삭제했다가 다시 생성한 경우 연결이 끊기게 된다.
    이런 상황이 발생했을 경우 ESB 시스템에서 Listener를 재연결 해서 에러발생을 줄이도록 하기 위한 Schedule 이다.

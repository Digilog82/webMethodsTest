IS Memory Object > Running Local Server Socket Screen
Screen Heading: IS Administrator > Solutions > Socket Admin > IS Memory Object > Running Local Server Socket
Integration Server Memory에 저장하여 관리하는 Object 중에서 현재 실행중인 Local Socket Server의 목록을 조회하는 메뉴이다.
  • IS Memory Object > Running Local Server Socket 클릭 : 목록을 조회한다.
Search Screen
Note:  
  • Server Socket > Local Server Socket 메뉴에서 Port를 Open 하면 IS Memory에 Socket 정보가 저장되고 IS Memory Object > Running Local Server Socket 메뉴에서 조회할 수 있다.
    Port를 Close 하면 IS Memory에서 Socket 정보가 삭제되고 IS Memory Object > Running Local Server Socket 메뉴에서도 삭제된다.

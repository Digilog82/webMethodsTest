KFTC Online > Online Connection Screen
Screen Heading: IS Administrator > Solutions > Socket Admin > KFTC Online > Online Connection
Asynch Online Socket Client에 대한 Socket 연결 및 연결해제 처리를 하는 메뉴이다.
  • KFTC Online > Online Connection 클릭 : 목록을 조회한다.
  • To Do 필드의 "Connect" 링크 클릭 : Socket Server에 연결한다.
  • To Do 필드의 "Close" 링크 클릭 : Socket Server에서 연결해제한다.
  • All Connect 클릭 : 목록에 조회된 모든 Asynch Online Socket Client에 대해서 Socket Server에 연결한다.
  • All Close 클릭 : 목록에 조회된 모든 Asynch Online Socket Client에 대해서 Socket Server에서 연결해제한다.
Connect And Close Screen
Note:  
  • KFTC는 예를 들어서 설명하기 위한 System Name 이다.
  • Basic Info > Business Name 메뉴에서 Asynch Online Socket YN 필드가 'Yes'인 Business Name이 존재할 경우 Provider System Name 단위로 "ProviderSystemName Online" 이라는 메뉴가 생성된다.
  • Provider System Name이 'KFTC'인 모든 Asynch Online Socket Client의 Socket Name이 Display 된다.
  • Socket Server에 연결성공하면 Socket 관련 Object 정보를 IS Memory에 저장하고 연결상태를 계속 유지하며 데이터 수신 Socket인 경우에는 Listener Service가 실행된다.
  • Socket Server에서 연결해제하면 Socket 관련 Object 정보를 IS Memory에서 삭제하고 데이터 수신 Socket인 경우에는 Listener Service가 종료된다.
  • Server Socket > Remote Server Socket 메뉴에서 Allowed Session Count 필드에 값을 설정한 경우 IP/Port에 대한 연결갯수가 이 값을 초과하지 못하도록 제한한다.
  • 연결에 문제가 발생 시(Server 쪽에서 Socket Server를 Restart 하는 경우도 포함) JSocketAdapter에서 자동으로 Socket Server에 재연결 하고 ESB 관리자에게 메일로 통보한다.
  • Socket Name은 Asynch Online Socket Business Name 단위로 생성한다.
  • 운영서버가 두 대인 경우 한 쪽 운영서버에서만 Socket Server에 연결하도록 한다.

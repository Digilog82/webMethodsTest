Utility > Reserve Project Deployment
Screen Heading: IS Administrator > Solutions > Socket Admin > Utility > Reserve Project Deployment
IS Administrator > Solutions > Deployer 메뉴에서 관리하는 Project에 대해서 Deployment Schedule을 등록하는 메뉴이다.
  • Utility > Reserve Project Deployment 클릭 : 목록을 조회한다.
  • Reserve Project Deployment 클릭 : 선택한 Project를 Deployment 하는 Schedule을 등록한다.
Search Screen
Note:  
  • IS Administrator > Solutions > Deployer 메뉴에서 등록한 모든 Project 정보가 조회된다.
  • Deployer 메뉴에서 Define, Build, Map 탭의 모든 기능을 미리 수행하고 Deploy 탭에서는 "Create Deployment Candidate" 만 미리 수행한다. 등록한 Schedule이 실행될 때 Deploy 탭의 "Checkpoint", "Deploy" 기능이 수행된다.
  • Check Box는 적어도 하나 이상 선택해야 한다.
  • 만약, Source Deploy가 야간에 수행될 경우에 유용하게 사용할 수 있는 기능이다.
  • Deployment 예약을 하면 정해진 시간에 Deployment 되도록 Schedule 등록을 하는 것으로서 Utility > Deployment Reservation 메뉴에서 [조회/수정/삭제] 할 수 있다.
    이 Schedule은 IS Administrator > Server > Scheduler 메뉴에서도 확인할 수 있다.
    이 Schedule은 한 번만 실행되는 것으로서 실행되고 나면 목록에서 자동으로 삭제된다.
    이 Schedule에 의해서 Deployment 처리를 한 후에 Deployment 결과를 ESB 관리자에게 메일로 통보한다.
Reservation Date
Schedule이 실행될 날짜를 입력한다.
Reservation Time
Schedule이 실행될 시간을 입력한다.

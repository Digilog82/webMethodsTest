IS Memory Object > Online Response Data Screen
Screen Heading: IS Administrator > Solutions > Socket Admin > IS Memory Object > Online Response Data
Integration Server Memory에 저장하여 관리하는 Object 중에서 응답전문 정보 목록을 조회하는 메뉴이다.
  • IS Memory Object > Online Response Data 클릭 : 목록을 조회한다.
  • Delete 아이콘 클릭 : Online Response Data 정보를 삭제한다.
Search Screen
Note:  
  • Asynch Online Socket Client가 존재하는 경우에만 이 메뉴가 Display 된다.
  • Asynch Online Socket의 경우 요청전문 송신과 응답전문 수신이 Asynch 방식으로 처리되기 때문에 요청전문 송신 후 응답전문을 IS Memory 또는 파일 시스템에서 구하는데 응답전문 수신 서비스에서는 응답전문을 IS Memory 또는 파일 시스템에 저장한다.
    Basic Info > Custom Variable 메뉴에서 ESB.SOCKET.ONLINE.SAVE의 설정에 따라서 IS Memory에 저장할지 파일 시스템에 저장할지 결정되는데 여기서는 IS Memory, 파일 시스템에 저장되어 있는 정보 모두 조회된다.
  • 전문 송수신 인터페이스가 완료되면 Online Response Data 정보는 삭제를 하기 때문에 일반적으로는 조회되는 정보가 없다. 그러나, 전문 송수신 인터페이스가 수행되는 바로 그 순간에는 잠시 조회될 수도 있다.
    전문 송수신 인터페이스가 완료된 후에도 예외상황 발생으로 인해서 Online Response Data 정보가 삭제되지 않을 수 있다. 이런 경우에는 오류 원인을 파악한 후에 ESB 관리자가 직접 삭제를 할 수 있다.
  • IS Memory Object > Online Data Delete Schedule 메뉴에서 Schedule을 등록해서 보관일수가 지난 데이터는 자동으로 삭제되도록 할 수 있다.

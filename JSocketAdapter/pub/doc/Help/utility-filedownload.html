Utility > Download File Screen
Screen Heading: IS Administrator > Solutions > Socket Admin > Utility > Download File
ESB Local 파일시스템에 존재하는 파일을 PC Local 파일시스템에 Download 하는 메뉴이다.
  • Utility > Download File 클릭 : Download 처리 화면으로 이동한다.
  • Search File 클릭 : Download Path에 있는 파일 목록을 조회한다.
  • Download Files 버튼 클릭 : 선택한 파일들을 PC Local 파일시스템에 Download 한다.
  • Delete Files 버튼 클릭 : 선택한 파일들을 ESB Local 파일시스템에서 삭제한다.
  • Download 버튼 클릭 : 해당 파일을 PC Local 파일시스템에 Download 한다.
  • Delete 버튼 클릭 : 해당 파일을 ESB Local 파일시스템에서 삭제한다.
Download Screen
Note:  
  • ESB Server Local 파일시스템에 존재하는 파일을 PC의 Local 파일시스템에 Download 할 수 있다.
    PC에서 ESB Server에 FTP 또는 SFTP 연결이 안 되는 경우에 유용하게 사용할 수 있다.
  • Check Box를 두 개 이상 선택해서 "Download Files" 버튼을 클릭하면 선택된 파일들을 동시에 다운로드할 수 있다.
    그러나, MS IE에서는 정상적으로 동작하지 않고 MS Edge, Chrome에서는 정상적으로 동작한다.
  • Download 버튼을 클릭하면 해당 파일 하나만 다운로드 한다.
  • ESB Server Local 파일시스템에 존재하는 파일을 삭제할 수 있다.
  • Check Box를 두 개 이상 선택해서 "Delete Files" 버튼을 클릭하면 선택된 파일들을 동시에 삭제할 수 있다.
  • Delete 버튼을 클릭하면 해당 파일 하나만 삭제한다.

AS/400 > Toggle Listener Screen
Screen Heading: IS Administrator > Solutions > Socket Admin > AS/400 > Toggle Listener
AS/400 > Data Q Listener 메뉴에 등록되어 있는 Listeners에 대해서 현재 서버에서 일괄적으로 Enable 또는 Disable 처리하는 메뉴이다.
  • AS/400 > Toggle Listener 클릭 : 목록을 조회한다.
  • Search Listener 클릭 : 조회조건에 해당하는 Listeners를 조회한다.
  • Enable Listener 클릭 : 조회된 Listeners에 대해서 일괄적으로 Enable 처리를 한다.
  • Disable Listener 클릭 : 조회된 Listeners에 대해서 일괄적으로 Disable 처리를 한다.
Search And Toggle Screen
Note:  
  • AS/400 메뉴는 AS400 시스템이 존재하고 Data Queue를 이용해서 ESB와 연계하는 경우에만 Display 된다.
    즉, Basic Info > Custom Variable 메뉴에서 ESB.CONFIG.AS400.DATAQUSEYN의 Value가 true로 설정되어 있는 경우에만 Display 된다.
  • AS/400 메뉴는 IS Administrator > Adapters > AS/400 Adapter에서 제공되지 않는 Data Queue Listener 관리에 대한 편리 기능을 제공한다.
    이런 기능을 사용하기 위해서는 IS Administrator > Adapters > AS/400 Adapter > Data Queue Listeners 메뉴에서 등록한 Listener는 AS/400 > Data Q Listener 메뉴에서도 모두 등록하도록 한다.
  • 여기에서 Enable, Disable 처리를 하면 IS Administrator > Adapters > AS/400 Adapter > Data Queue Listeners 메뉴에 등록되어 있는 Listeners에 적용되는 것이다.

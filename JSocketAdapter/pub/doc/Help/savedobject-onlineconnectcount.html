IS Memory Object > Online Connect Count Screen
Screen Heading: IS Administrator > Solutions > Socket Admin > IS Memory Object > Online Connect Count
Integration Server Memory에 저장하여 관리하는 Object 중에서 현재 서버에서 연결중인 Asynch Online Socket Client의 연결갯수를 IP/Port 별로 목록을 조회하는 메뉴이다.
  • IS Memory Object > Online Connect Count 클릭 : 목록을 조회한다.
Search Screen
Note:  
  • Asynch Online Socket Client가 존재하는 경우에만 이 메뉴가 Display 된다.
  • KFTC Online > Online Connection 메뉴에서 Remote Socket Server에 연결하면 IS Memory에 연결갯수를 하나 증가한 정보가 저장되고 IS Memory Object > Online Connect Count 메뉴에서 조회할 수 있다.
    연결을 해제하면 IS Memory에서 연결갯수를 하나 뺀 정보가 저장되는데 연결갯수가 0이면 IS Memory에서 삭제되고 IS Memory Object > Online Connect Count 메뉴에서도 삭제된다.
  • Server Socket > Remote Server Socket 메뉴에서 관리하는 Online Client에 대한 Allowed Session Count 필드에 값이 설정된 경우에만 연결 갯수를 관리하고
    IS Memory Object > Online Connect Count 메뉴에 Display 된다.

Deploy Config > Queue/Topic
Screen Heading: IS Administrator > Solutions > Socket Admin > Deploy Config > Queue/Topic
UM에 생성한 Queue 또는 Topic 정보를 다른 서버에 Deploy 하는 메뉴이다.
  • Deploy Config > Queue/Topic 클릭 : 목록을 조회한다.
  • Deploy Queue/Topic 클릭 : 선택한 Queue/Topic을 Target Server에 Deploy 한다.
Search And Deploy Screen
Note:  
  • Deploy Config 메뉴는 개발서버, UAT서버에서는 Display 되고 운영서버에서는 Display 되지 않는다.
  • Check Box를 적어도 하나 이상 선택해야 한다.
  • Target Server를 적어도 하나 선택해야 Deploy 가능하다.
  • Queue 또는 Topic이 Target Server에 이미 생성되어 있는 경우에는 Target Server에서 생성을 생략한다.
Target Server
Deploy 하고자 하는 Target Server
Basic Info > Custom Variable 메뉴의 ESB.CONFIG.REMOTE.DEPCONFIG Variable에 등록된 Remote Server Alias 목록

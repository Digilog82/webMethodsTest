IS Memory Object > Online Reconnecting Server Screen
Screen Heading: IS Administrator > Solutions > Socket Admin > IS Memory Object > Online Reconnecting Server
Integration Server Memory에 저장하여 관리하는 Object 중에서 Asynch Online Socket 재연결 중인 정보 목록을 조회하는 메뉴이다.
  • IS Memory Object > Online Reconnecting Server 클릭 : 목록을 조회한다.
  • Delete 아이콘 클릭 : Online Reconnecting Server 정보를 삭제한다.
Search Screen
Note:  
  • Asynch Online Socket Client가 존재하는 경우에만 이 메뉴가 Display 된다.
  • Asynch Online Socket 연결에 문제가 발생했을 경우 JSocketAdapter에서는 기존의 연결을 해제하고 재연결 하도록 되어 있는데 여러 개의 서비스에서 동시에 재연결하는 것을 방지하기 위해서 재연결 시작할 때 재연결 표시정보를 IS Memory에 저장해서 다른 서비스에서는 재연결 처리하지 못 하도록 하고 있다. 재연결 완료 후에 Online Reconnecting Server 정보는 IS Memory에서 삭제 된다.
  • 재연결이 완료되면 Online Reconnecting Server 정보는 삭제를 하기 때문에 일반적으로는 조회되는 정보가 없다. 그러나, 재연결이 수행되는 바로 그 순간에는 잠시 조회될 수도 있다.
    재연결이 완료된 후에도 예외상황 발생으로 인해서 Online Reconnecting Server 정보가 삭제되지 않을 수 있다. 이런 경우에는 오류 원인을 파악한 후에 ESB 관리자가 직접 삭제를 할 수 있다.
  • IS Memory Object > Online Data Delete Schedule 메뉴에서 Schedule을 등록해서 보관일수가 지난 데이터는 자동으로 삭제되도록 할 수 있다.

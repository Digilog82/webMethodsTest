Utility > Server Log Screen
Screen Heading: IS Administrator > Solutions > Socket Admin > Utility > Server Log
Integration Server 로그파일을 날짜별로 조회하는 메뉴이다.
  • Utility > Server Log 클릭 : 조회화면으로 이동한다.
  • Search Logs 클릭 : 조회조건에 해당하는 로그파일을 조회한다.
  • Download File 클릭 : 조회조건에 해당하는 로그파일을 Download 한다.
Search Screen
Note:  
  • SAG_Directory/logs 아래에 있는 server.log 파일을 날짜별로 조회할 수 있다.
    IS Administrator > Logs > Server 메뉴에서는 오늘 날짜의 로그만 조회가 가능하다.
    이 메뉴에서 오늘 날짜 뿐만 아니라 과거 날짜의 서버로그도 조회가 가능하다.
  • Basic Info > Delete Local File 메뉴에서 SAG_Directory/logs 디렉토리를 등록해서 보관일수가 지난 파일은 자동으로 삭제되도록 할 수 있다.
Download Threshold
로그파일을 조회할 때 파일 사이즈가 너무 큰 경우에는 서버 성능에 영향을 줄 수 있다.
Download Threshold에 설정한 값보다 파일 사이즈가 클 경우에는 로그파일을 조회하지 않고 자동으로 파일을 Download 하도록 되어 있다.
즉, Download Threshold에 설정한 값보다 파일 사이즈가 작을 경우에 로그파일을 조회해서 화면에 Display 한다.

Utility > Check Target Connection Screen
Screen Heading: IS Administrator > Solutions > Socket Admin > Utility > Check Target Connection
Target Server와의 방화벽 오픈여부 및 Server Running 상태를 Check하는 메뉴이다.
  • Utility > Check Target Connection 클릭 : 테스트 화면으로 이동한다.
  • Test Connection 클릭 : Server와의 연결 테스트를 수행하고 테스트 결과를 Display 한다.
Test Screen
Note:  
  • ESB Server와 Target Server와의 방화벽 오픈여부 및 Target Server의 Running 상태를 알 수 있다.
  • 만약에 방화벽이 오픈되어 있지 않다면 Connection Timeout이 발생할 때까지 기다리는 시간이 길므로(몇분 소요) Target Host Properties 영역 Connect Timeout 필드의 값을 조정해서 기다리는 시간을 줄일 수 있다.
    Connect Timeout 필드에 설정한 시간 안에 연결이 안되면 Connection Timeout이 발생하고 테스트를 종료한다.
  • Connection Test Results 영역 Error Message 필드에 "Connection refused" 라는 문구가 포함되어 있으면 Target Server가 Shutdown 상태인 것이다.
  • Target Server가 Online Socket Server일 경우에는 연결 갯수 제한 등의 설정으로 Target Server에 장애가 발생할 수 있으므로 Online Socket Server와는 Connection 테스트를 수행하지 않도록 주의해야 한다.

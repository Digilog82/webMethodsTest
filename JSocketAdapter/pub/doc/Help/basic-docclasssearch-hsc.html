Basic Info > Doc Class Search Screen
Screen Heading: IS Administrator > Solutions > Socket Admin > Basic Info > Doc Class Search
Local Socket Server가 제공하는 Business에 대한 전문 분류별 전문 목록을 조회하는 메뉴이다.
  • Basic Info > Doc Class Search 클릭 : 목록을 조회한다.
  • Search 버튼 클릭 : 조회조건에 해당하는 전문 목록을 조회한다.
Search Screen
Note:  
  • 조회 조건을 하나도 선택하지 않으면 전체 목록을 조회한다.
  • 조회 조건은 두 개 이상 선택 가능하다.
Common Header Length
전문의 공통헤더 길이별로 전문 목록을 조회한다.
Listening Port
EAI 시스템에서 전문을 수신하는 Port 번호별로 전문 목록을 조회한다.
Health Check Doc YN
Health Check 전문인지 일반전문인지에 대한 구분별로 전문 목록을 조회한다.
ANB Return Doc YN
EAI 시스템에서 전문 수신 후 Source 시스템에 ANB 전문을 리턴해야 하는 전문인지 아닌지에 대한 구분별로 전문 목록을 조회한다.
Target Socket Name
전문을 송신해야 하는 Target Socket Name별로 전문 목록을 조회한다.
One Way Doc YN
One Way 방식의 전문인지 아닌지에 대한 구분별로 전문 목록을 조회한다.
One Way Port
One Way 방식의 전문인 경우에 전문을 송신하는 Port 번호별로 전문 목록을 조회한다.
Doc Length Type
전문길이 유형별로 전문 목록을 조회한다.
Source Target System
전문의 송수신 시스템(Source System --> Target System) 별로 전문 목록을 조회한다.
Serial Processing Queue/Topic Use YN
전문을 순차처리하기 위한 Queue/Topic을 사용하는지 안하는지에 대한 구분별로 전문 목록을 조회한다.
"No"를 선택해서 조회되는 전문은 순차처리를 하지 않는 전문이다.
Sending Doc Length Change YN
전문을 Target System에 송신하기 전에 공통헤더의 전문길이 필드의 값을 변경하는지 안하는지에 대한 구분별로 전문 목록을 조회한다.

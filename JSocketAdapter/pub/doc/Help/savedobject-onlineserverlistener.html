Online Server > Online Server Listener Screen
Screen Heading: IS Administrator > Solutions > Socket Admin > Online Server > Online Server Listener
Integration Server Memory에 저장하여 관리하는 Object 중에서 현재 실행중인 Online Server Listener 목록을 조회하는 메뉴이다.
  • Online Server > Online Server Listener 클릭 : 목록을 조회한다.
  • Delete 아이콘 클릭 : Listener Service를 종료한다.
Search Screen
Note:  
  • Asynch Online Socket Server가 존재하는 경우에만 이 메뉴가 Display 된다.
  • Asynch Online Server Socket에 Listener Service가 설정되어 있으면서 Client가 연결하는 순간에 Listener Service가 실행되고 이 서비스에 대한 정보를 IS Memory에 저장한다.
    Client에서 Socket을 Close 하든가 Server에서 Socket을 Close 하면 Listener Service가 종료되고 IS Memory에서 이 서비스에 대한 정보를 삭제한다.
  • Listener Service를 종료하고 해당 세션에 대한 Client 연결을 강제로 끊을 수 있다.
    네트웍 단절 등 Client와의 연결이 비정상적으로 종료된 경우에는 Listener Service에서 Client와의 연결 끊김을 알 수 없어서 Listener Service가 계속 실행중 상태로 있고 Client와의 연결도 유효한 것으로 인식을 한다. 이런 경우에 이 기능을 사용할 수 있다.
